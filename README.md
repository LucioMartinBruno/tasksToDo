Electron taskToDO Desktop App

This is a cross platform taskToDo desktop application built on electron

Usage
Installation

Install the dependencies

$ npm install

Serve

To run electron

$ npm start

To Package & Build

For Windows

$ npm run package-win

For Mac

$ npm run package-mac

For Linux

$ npm run package-linux
